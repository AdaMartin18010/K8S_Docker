## 容器设计模式

| 模式 | 用途 | 示例 |
|------|------|------|
| Sidecar | 日志收集、监控代理 | Fluent Bit + App |
| Ambassador | 代理外部服务 | Redis 代理 |
| Adapter | 标准化接口 | 指标转换 |
| Init | 初始化任务 | 数据库迁移 |

## 部署模式

| 模式 | 特点 | 适用场景 |
|------|------|----------|
| 滚动更新 | 渐进替换 | 常规发布 |

## 弹性模式

- **熔断器**: 防止级联故障
- **重试**: 自动重试失败请求
- **限流**: 控制请求速率
- **超时**: 避免长时间等待
