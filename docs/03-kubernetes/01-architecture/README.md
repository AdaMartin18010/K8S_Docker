## 2025 年重要更新

| 版本 | 特性 | 影响 |
|------|------|------|
| 1.30 | Sidecar 容器 GA | 原生支持 Sidecar 生命周期 |
| 1.31 | CephFS/RBD 移除 | 必须使用 CSI 驱动 |
| 1.32 | 内存管理器 GA | Guaranteed Pod 独占内存 |
| 1.32 | QueueingHint | 调度器吞吐量优化 |
