## 架构全景

```
┌─────────────────────────────────────────────────────────────┐
│                    控制平面 (Control Plane)                   │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────────────┐  │
│  │ API Server  │  │   etcd      │  │ Controller Manager  │  │
│  │  (kube-     │  │  (存储)      │  │   (控制器)          │  │
│  │ apiserver)  │  │             │  │                     │  │
│  └──────┬──────┘  └─────────────┘  └─────────────────────┘  │
│         │                                                   │
│  ┌──────┴────────────────────────────────────────────────┐  │
│  │              Scheduler (调度器)                        │  │
│  └────────────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────────┘
                              │
┌─────────────────────────────┼───────────────────────────────┐
│                     数据平面 (Data Plane)                   │
│  ┌──────────────────────────┴───────────────────────────┐   │
│  │                    Worker Node                        │   │
│  │  ┌─────────────┐  ┌─────────────┐  ┌───────────────┐ │   │
│  │  │   Kubelet   │  │  Kube-proxy │  │    Runtime    │ │   │
│  │  │ (节点代理)  │  │  (网络代理) │  │(containerd/   │ │   │
│  │  │             │  │             │  │ CRI-O)        │ │   │
│  │  └─────────────┘  └─────────────┘  └───────────────┘ │   │
│  └─────────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────┘
```

## 核心组件职责

### 控制平面

| 组件 | 版本 | 2025年更新 |
|------|------|-----------|
| kube-apiserver | 1.32 | 内存管理器 GA |
| etcd | 3.5+ | QueueingHint 优化调度 |
| kube-scheduler | 1.32 | 吞吐量提升 |
| kube-controller-manager | 1.32 | 稳定性增强 |

### 数据平面

| 组件 | 职责 |
|------|------|
| Kubelet | 管理 Pod 生命周期 |
| Kube-proxy | 实现 Service 网络 |
| Container Runtime | 运行容器 (containerd/CRI-O) |

## 快速开始

```bash
# 查看集群版本
kubectl version

# 查看节点
kubectl get nodes -o wide

# 查看组件状态
kubectl get componentstatuses

# 查看 Pod (所有命名空间)
kubectl get pods -A
```
