## 安全层次

```
┌─────────────────────────────────────────┐
│  1. 供应链安全                           │
│     - 镜像签名 (Cosign)                  │
│     - SBOM 清单                          │
│     - 漏洞扫描 (Trivy)                   │
├─────────────────────────────────────────┤
│  2. 运行时安全                           │
│     - 沙箱运行时 (gVisor/Kata)           │
│     - Seccomp/AppArmor                   │
│     - Pod Security Standards             │
├─────────────────────────────────────────┤
│  3. 网络安全                             │
│     - NetworkPolicy                      │
│     - 零信任架构                         │
│     - mTLS 通信                          │
├─────────────────────────────────────────┤
│  4. 访问控制                             │
│     - RBAC                               │
│     - 准入控制器                         │
│     - 密钥管理                           │
├─────────────────────────────────────────┤
│  5. 合规审计                             │
│     - 审计日志                           │
│     - 合规检查                           │
│     - 安全监控                           │
└─────────────────────────────────────────┘
```
